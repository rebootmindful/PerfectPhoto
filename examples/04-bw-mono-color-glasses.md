# 案例4：黑白单色+彩色眼镜

> 标签：`棚拍` `黑白` `单色点缀` `编辑感` `戏剧性光线` `男性`
> 成立点：黑白底色+彩色眼镜点缀=编辑感高级视觉
> 适用步骤：步骤2成像方式, 步骤7光线, 步骤8镜头

## Prompt

```
3:4竖图，真实摄影，黑白单色点缀写真
黑白胶片摄影，高对比度，颗粒感

一位明确成年的东亚男性，棱角分明，短发利落
黑白妆容，哑光底妆，深色眉形
黑色西装外套，戴彩色透明框眼镜（画面唯一彩色元素）
纯灰背景棚拍，硬光从侧方打入
前景有虚化的几何色块，人物正面微侧，背景是深色幕布
中景半身，硬光阴影，高对比，编辑感构图
人物正面微侧头，眼神直视镜头，肩膀一高一低
侧方硬光，脸部一半亮一半暗，强阴影
不要塑料皮肤，不要手指畸形，不要多余肢体
```

## 成立点分析

这张照片成立在「单色点缀」——黑白画面中唯一彩色的眼镜成为视觉锚点。硬光+高对比+编辑感构图，三者互相认识。
