# 案例8：金色庆祝棚拍

> 标签：`棚拍` `金色` `庆祝` `高调` `全身` `暖调` `女性`
> 成立点：金色光线+庆祝动作+高调棚拍=欢乐氛围感
> 适用步骤：步骤2成像方式, 步骤5服装, 步骤7光线

## Prompt

```
3:4竖图，真实摄影，金色庆祝棚拍写真
数码单反摄影，高调亮调，金色暖调

一位明确成年的东亚女性，开朗气质，棕色长发飘逸
金色系妆容，光泽底妆，香槟色眼影，裸粉唇
金色亮片吊带裙，金色耳环，手持香槟杯
纯白背景棚拍，金色光线从上方和侧方打出
前景有飘落的金色亮片，人物跳跃庆祝，背景是白色无缝背景纸
全身，高调亮调，金色暖调
人物跳跃庆祝，双手举起，头发飞扬
金色光线从上方和侧方打出，皮肤有光泽，亮片闪光
不要塑料皮肤，不要手指畸形，不要多余肢体
```

## 成立点分析

这张照片成立在「庆祝氛围」——金色光线+跳跃动作+亮片，三者共同构建了欢乐的庆祝瞬间。高调棚拍让画面明亮通透。
