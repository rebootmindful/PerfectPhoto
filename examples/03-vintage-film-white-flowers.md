# 案例3：复古胶片与白花

> 标签：`户外` `复古` `胶片` `花朵` `道具` `低机位` `文艺感` `女性` `半身`
> 成立点：胶片颗粒+白色花朵+低机位=文艺复古氛围
> 适用步骤：步骤3场景, 步骤5服装, 步骤6动作, 步骤8镜头

## Prompt

```
3:4竖图，真实摄影，复古胶片白花写真
富士相机摄影，低饱和胶片色调，明显颗粒感

一位明确成年的东亚女性，文艺气质，黑色长发自然披散
清透淡妆，自然眉形，裸色唇
白色棉麻衬衫，米色长裙，手持白色雏菊
户外草地，午后柔和自然光，背景有虚化的白色花丛
前景有虚化的白色花朵和绿叶，人物坐在草地上，背景是延伸的花丛
低机位，镜头微仰，近景半身，浅景深
人物低头闻手中花朵，头肩方向一致，不强求看镜头
午后柔和自然光，侧逆光打在发丝上，花瓣柔焦
不要塑料皮肤，不要手指畸形，不要多余肢体
```

## 成立点分析

这张照片成立在「文艺复古感」——胶片颗粒+白花+低机位仰拍，三者共同构建了复古文艺的氛围。低机位让花朵成为前景遮挡，画面有层次。
