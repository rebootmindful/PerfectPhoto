# 案例7：后台化妆间复古街拍

> 标签：`室内` `后台` `化妆灯` `街头感` `复古` `女性`
> 成立点：后台场景密度+化妆灯+复古街拍感=有故事的画面
> 适用步骤：步骤3场景, 步骤5服装, 步骤7光线

## Prompt

```
3:4竖图，真实摄影，后台化妆间复古街拍写真
胶片相机摄影，复古色调，暖色颗粒感

一位明确成年的东亚女性，复古气质，黑色波浪卷发
复古红唇妆，哑光底妆，上挑眼线，正红唇色
复古碎花连衣裙，珍珠项链，黑色高跟鞋
后台化妆间，化妆灯从镜子两侧打出柔和光，背景有衣架和服装
前景有虚化的化妆刷和镜子，人物坐在化妆镜前，背景是忙碌的后台
中景半身，化妆灯柔光，复古色调
人物侧身整理耳环，头肩方向一致，眼神看向镜子
化妆灯从镜子两侧打出柔和光，脸部均匀柔和，暖色调
不要塑料皮肤，不要手指畸形，不要多余肢体
```

## 成立点分析

这张照片成立在「场景密度」——后台化妆间不是一个简单的背景，而是有故事的场景。化妆灯+复古妆+衣架服装，三者构建了「演出前的瞬间」的叙事感。
