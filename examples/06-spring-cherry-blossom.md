# 案例6：春日樱花粉色浪漫

> 标签：`户外` `春日` `樱花` `粉色` `浪漫` `85mm` `雨天` `女性`
> 成立点：色彩统一（全画面粉色系）+ 湿润质感 + 85mm虚化
> 适用步骤：步骤3场景, 步骤5服装, 步骤8镜头

## Prompt

```
3:4竖图，真实摄影，春日樱花粉色浪漫写真
85mm f/1.4镜头，柔焦虚化，粉色色调

一位明确成年的东亚女性，甜美气质，棕色长发微卷
粉色系妆容，粉嫩底妆，粉桃腮红，水润唇
浅粉色风衣，白色内搭，漆皮小包
樱花树下，春日阴天柔和光线，地面湿润有积水
前景有虚化的樱花枝条，人物站在樱花树下，背景是延伸的樱花道
近景半身，f/1.4浅景深，背景大面积虚化
身体微侧向镜头，头部转向观众（同向轻微侧身），手接住飘落的花瓣
春日阴天柔和光线，湿润地面反射粉色，漆皮包有光泽质感
不要塑料皮肤，不要手指畸形，不要多余肢体
```

## 成立点分析

色彩统一是灵魂——从风衣到花到樱花到路面落花，全画面粉色系。湿润地面的反射+漆皮的光泽=质感层次。85mm f/1.4 虚化背景但保留樱花氛围。注意：「身体微侧向镜头，头部转向观众」是同向轻微侧身，不是大幅扭头，和动作骨架原则不冲突。
