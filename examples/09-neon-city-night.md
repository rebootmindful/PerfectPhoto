# 案例9：霓虹都市夜拍

> 标签：`夜景` `都市` `霓虹` `戏剧性光线` `男性` `低角度`
> 成立点：霓虹光线染色+低角度仰拍+力量感=都市夜归人
> 适用步骤：步骤7光线, 步骤8镜头

## Prompt

```
3:4竖图，真实摄影，霓虹都市夜拍写真
数码单反摄影，高对比度，青橙对比色调

一位明确成年的东亚男性，酷帅气质，黑色短发
自然妆容，清透底妆，自然眉形
深色羽绒服，亮橙色毛衣内搭
明亮的橙黄色霓虹灯城市隧道，夜晚
前景有虚化的霓虹灯光晕，人物站在隧道中央，背景是延伸的霓虹灯
低角度仰拍，高对比，青橙对比色调
人物低头看镜头，肩膀放松，手插口袋
霓虹灯从两侧打出暖色光，脸部被橙黄光染色，低角度仰拍增加力量感
不要塑料皮肤，不要手指畸形，不要多余肢体
```

## 成立点分析

霓虹光线是灵魂——橙黄霓虹+青橙对比色调，整个画面被光染色。低角度仰拍+低头看镜头=力量感。羽绒服和毛衣的暖色和霓虹暖光认识。
