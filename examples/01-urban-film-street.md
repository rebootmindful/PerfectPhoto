# 案例1：都市胶片街拍

> 标签：`户外` `都市` `胶片` `自然光` `日常感` `男性` `半身`
> 成立点：胶片颗粒+自然街景+日常动作=真实抓拍感
> 适用步骤：步骤2成像方式, 步骤3场景, 步骤7光线

## Prompt

```
3:4竖图，真实摄影，都市胶片街拍写真
富士相机摄影，高饱和胶片色调，轻微颗粒感

一位明确成年的东亚男性，干净利落气质，短发微乱
自然淡妆，清透底妆，自然眉形
深色工装夹克，白色T恤，简约银链
城市街头，午后自然光，背景有虚化的老建筑和行人
前景有虚化的树叶和电线杆，人物站在街边，背景是延伸的街道
近景半身，浅景深，大胆裁切
人物侧身看远方，肩膀放松，手插口袋
午后自然光，侧光打在脸上，建筑投下柔和阴影
不要塑料皮肤，不要手指畸形，不要多余肢体
```

## 成立点分析

这张照片成立在「日常感」——不是摆拍，而是像被人抓拍的瞬间。胶片颗粒+自然街景+手插口袋的动作，三者互相认识。
