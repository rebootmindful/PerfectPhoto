# 案例5：门口暖阳生活感

> 标签：`室内` `窗光` `温柔` `生活化` `自然光` `情绪感` `女性`
> 成立点：窗光+生活化场景+温柔情绪=真实生活瞬间
> 适用步骤：步骤3场景, 步骤6动作, 步骤7光线

## Prompt

```
3:4竖图，真实摄影，门口暖阳生活感写真
胶片相机摄影，暖色调，柔和颗粒感

一位明确成年的东亚女性，温柔气质，棕色长发松散扎起
自然裸妆，奶油肌，淡腮红，自然唇色
米色针织毛衣，宽松舒适
家门口门廊，午后暖阳从窗户斜射进来，背景有虚化的室内陈设
前景有虚化的门框和绿植，人物靠在门框上，背景是温暖的室内
近景半身，柔和光线，浅景深
人物微低头看书，手捧一杯热茶，肩膀放松
午后暖阳从窗户斜射，脸部被柔和光照亮，温暖光斑
不要塑料皮肤，不要手指畸形，不要多余肢体
```

## 成立点分析

这张照片成立在「生活感」——不是摆拍，而是真实的生活瞬间。窗光+门廊+看书喝茶，三者构建了温暖的日常氛围。
