# 案例2：赛博霓虹棚拍

> 标签：`棚拍` `赛博` `霓虹` `戏剧性光线` `情绪感` `女性`
> 成立点：霓虹光线染色+硬光阴影+赛博美学=科技感氛围
> 适用步骤：步骤2成像方式, 步骤7光线, 步骤8镜头

## Prompt

```
3:4竖图，真实摄影，赛博霓虹棚拍写真
数码单反摄影，高对比度，霓虹色彩

一位明确成年的东亚女性，酷飒气质，黑色短发挑染银色
金属感妆容，冷白底妆，深灰眼影，哑光暗唇
黑色皮夹克，银色金属配饰，霓虹色耳钉
纯黑背景棚拍，霓虹灯管从两侧打出彩色光
前景有虚化的霓虹灯管光晕，人物正面站立，背景是深色幕布
中景半身，硬光阴影，高对比
人物正面微低头，眼神锐利看镜头，肩膀平端
霓虹灯管从两侧打出彩色光，脸部一半冷蓝一半暖橙，硬光阴影
不要塑料皮肤，不要手指畸形，不要多余肢体
```

## 成立点分析

这张照片成立在「光线染色」——整个画面被霓虹光染成赛博色调。硬光+高对比+金属配饰，三者互相认识。
