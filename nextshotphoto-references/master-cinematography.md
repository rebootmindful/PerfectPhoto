# 大师运镜库

选了大导演之后，基础运镜的变换规则不是全部照搬——大师的签名风格会覆盖/增强特定运镜的参数。

---

## 8 位大师预设

```
🎥 选一位电影导演跟拍（可选，不选直接走下一步）：

▸ 希区柯克 · 空间扭曲 — 推拉变焦（Dolly Zoom）制造心理压迫，
  适合震惊/恐惧/情绪崩塌瞬间
▸ 斯皮尔伯格 · 环绕移焦 — 半圆周运动 + 焦点切换，适合群像/对话/揭示
▸ 迈克尔·贝 · 低角旋转 — 低机位广角半环绕 + 慢动作英雄姿态，
  适合登场/决战/史诗感高潮
▸ 诺兰 · 推镜压迫 — IMAX 大画幅缓慢推镜 + 交叉剪辑，
  适合悬念/对峙/时间感
▸ 王家卫 · 手持迷离 — 轻微晃动 + 慢速快门 + 霓虹拖影，
  适合情绪/都市/夜，不完美=真实
▸ 库布里克 · 单点透视 — 笔直推轨 + 对称构图，
  适合走廊/仪式感/压抑/秩序
▸ 韦斯·安德森 · 横移平拍 — 90度正对横移 + 居中对称 + 俯拍，
  适合展示/空间/幽默/平面感
▸ 姜文 · 急推急拉 — 高速变焦 + 跳切 + 爆发力，
  适合能量感/喜剧/突然转折
```

## 大师风格对变换矩阵的微调

### 斯皮尔伯格（环绕移焦）

影响的基础运镜：环绕 (Orbit)
```
轨迹：        全圆 → 限定为半圆 180 度
焦点：        固定焦点 → 焦点在主体之间切换（移焦）
速度：        匀速 → 慢入慢出，起幅落幅各留 2 秒
追加描述：    "斯皮尔伯格式环绕移焦，镜头围绕人物做半圆平滑运动，
              焦点在[人物A]和[人物B]之间自然切换"
```

### 迈克尔·贝（低角旋转）

影响的基础运镜：环绕 (Orbit)
```
机位高度：    平视 → 低机位，仰拍角度
速度：        平滑 → 加速旋转，慢动作结束
追加描述：    "迈克尔·贝式低机位英雄环绕，广角镜头，
              以主体为中心快速半环绕，慢动作收尾"
```

### 诺兰（推镜压迫）

影响的基础运镜：推镜头 (Push In) + 拉镜头 (Pull Out)
```
推镜头增强：
  画幅：        常规 → IMAX 大画幅，全画幅无裁切
  速度：        平滑 → 极缓慢推进，制造时间拉伸感
  追加描述：    "诺兰式推镜，IMAX 全画幅，以极慢速度推进，
                每一帧都在强化压迫感"

拉镜头增强：
  目的：        展示环境 → 揭示时间/空间关系
  追加描述：    "回拉揭示全局，空间维度拉伸"
```

### 王家卫（手持迷离）

影响的基础运镜：手持晃动 (Handheld)
```
快门：        正常 → 慢速快门，拖影感
色彩：        正常 → 霓虹光溢出，色彩饱和度增强
追加描述：    "王家卫式手持，慢速快门轻微拖影，
              霓虹灯光在画面边缘形成绿色/红色溢出"
```

### 库布里克（单点透视）

影响的基础运镜：推镜头 (Push In)
```
机位：        任意 → 必须正面中轴，对称构图
轨迹：        可偏移 → 笔直推进，零偏移
追加描述：    "库布里克式单点透视，镜头沿中轴线笔直推进，
              画面严格对称"
```

### 韦斯·安德森（横移平拍）

影响的基础运镜：移镜头 (Track)
```
机位高度：    任意 → 必须 90 度正对主体
角度：        有角度 → 平行横移，零角度偏移
追加描述：    "韦斯·安德森式横移，镜头 90 度正对主体平行移动，
              居中对称构图"
```

### 姜文（急推急拉）

影响的基础运镜：推镜头 (Push In) + 拉镜头 (Pull Out)
```
推镜头增强：
  速度：        平滑 → 极速推进/骤停

拉镜头增强：
  速度：        平滑 → 瞬间拉出

追加描述：    "姜文式急推/急拉，镜头运动带爆发力，
              变焦结束瞬间定格"
```

## 大师风格与运镜的冲突规则

| 用户选的大师 | 用户选的运镜 | 行为 |
|-------------|-------------|------|
| 斯皮尔伯格 | 环绕 | 增强环绕，追加移焦 |
| 斯皮尔伯格 | 移镜头 | 保持标准变换 |
| 诺兰 | 推镜头 | 增强为 IMAX 慢推 |
| 诺兰 | 拉镜头 | 增强为空间揭示 |
| 王家卫 | 手持 | 增强，追加慢速快门 |
| 王家卫 | 推镜头 | 保持标准变换（王家卫不签名推镜） |
| 库布里克 | 推镜头 | 增强为单点透视推 |
| 库布里克 | 环绕 | 建议替换为单点推（库布里克风格不擅长环绕） |
| 韦斯·安德森 | 移镜头 | 增强为横移平拍 |
| 韦斯·安德森 | 推镜头 | 保持标准变换 |
| 姜文 | 推/拉 | 增强为急推急拉 |
| 姜文 | 移镜头 | 保持标准变换 |

**未选大师时**：使用标准变换矩阵，无任何增强/覆盖。

**选了大导演时**：变换矩阵加载对应大师的增强规则，替换/追加描述短语。同时 DIM 追问中的映射检查会加载大师专属的前置条件。例如选了斯皮尔伯格后环绕的映射检查会增加：「移焦切换的落点清楚吗？两个焦点之间的距离不能太远。」

## 运镜词库

以下词汇可植入 prompt 变体域的「运镜」字段：

**景别词**：远景 / 全景 / 中景 / 近景 / 特写 / 极特 / 过肩 / 双人 / 骑轴

**运动词**：推 / 拉 / 摇 / 移 / 跟 / 环绕 / 升降 / 甩 / 晃 / 穿越 / 俯冲 / 爬升 / 横滚 / 悬停 / 漂移 / 急停

**节奏词**：缓慢 / 平滑 / 快速 / 极速 / 匀速 / 变速 / 骤停 / 慢动作 / 抽帧

**辅助词**：动态模糊 / 呼吸感 / 视差 / 焦平面 / 浅景深 / 边缘虚化 / 透视压缩 / 空间扭曲

## 大师签名转场

每位大师的签���运镜伴���特定的转场方式。选大师后，转场类型自动对齐：

| 大师 | 签名转场 | 说明 |
|------|---------|------|
| 斯皮尔伯格 | 叠化 | 长叠化（1-2 秒），情绪过渡，环绕移焦配合叠化 |
| 诺兰 | 匹配剪辑 | 形状/动��匹配（圆形→圆形、门关→门开），时间跳跃感 |
| 迈克尔·贝 | 闪摇 | 快速甩镜转场 + 爆炸白光闪，能量感 |
| 王家卫 | 跳切 + 叠化 | 时间碎片化，不连贯剪辑 + 慢速叠化 |
| 库布里克 | 匹配剪辑 | 时空跳跃匹配（骨头→太空站），同形状跨时空连接 |
| 韦斯·安德森 | 横移转场 | 90度横移切换场景，画面如同一扇扇打开的门 |
| 姜文 | 急切/跳切 | 无过渡直接切换，爆发力，不讲道理的快 |
| 希区柯克 | 视线匹配 | 人物看→切到ta看的东西（主观视角），悬念感 |

**未选大师时**：默认硬切。
